# GulpConfiguration
Gulp的配置文件
</br>
配置出处：[写一份gulp常用配置文件，构建前端工作流](http://blog.csdn.net/qq_15096707/article/details/54293203)  

## 使用方法
安装依赖
```
cnpm install 
```

</br>初始化文件目录
```
gulp init 
```

</br>部署开发环境
```
npm start 或 npm dev 
```

</br>部署生产环境
```
npm build 
```

</br>清理dist文件夹
```
npm clean 
```
